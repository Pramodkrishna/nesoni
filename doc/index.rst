This is a cookbook of common uses of `Nesoni`_.
When a programmer sees a common usage pattern 
they say "I will turn that into a single step",
so most of these recipes can be achieved with one or two complex commands,
but it's useful to break this down so you know what it's doing and what the options are.
Therefore these recipes follow a general formula of:

#. Task as a single magic command
#. Task as individual steps
#. Task as a single command, revisited with more options

The "reference manual" for Nesoni is built into Nesoni itself.
For a list of available tools type:
::

   nesoni

For the parameters and flags for a particular tool type:
::

   nesoni <name-of-tool>:

.. _Nesoni: http://www.vicbioinformatics.com/software.nesoni.shtml
